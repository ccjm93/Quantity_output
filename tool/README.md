# 수량산출서 출력 자동화 도구 (단일 PDF 병합판)

토목 **수량산출서** 루트 폴더의 모든 시트를, **폴더·파일 순서대로 하나의 PDF**로 병합합니다.
각 폴더·파일 앞에는 **간지(divider)** 가 들어가고, 마지막으로 프로그램 내부 로직이
연속 `#` 2개 이상, `#REF!`, 빈 페이지, 하단 표 테두리 누락 의심과 그 **위치**를 알려줍니다.

## 동작 순서

1. **루트 폴더 지정** → 내부 폴더/파일 구조를 문서(자연) 순서로 파악
2. 각 시트에 **"페이지 나누어 미리보기" 활성화**
   - 인쇄영역은 **원래 지정된 값은 그대로**, 지정 안 된 시트는 **Excel 기본값**(자동)으로 둡니다.
   - **셀 크기는 절대 편집하지 않습니다.** 페이지 나누어 미리보기만 켭니다.
   - 이 설정은 **원본 파일에 저장**됩니다 → 처리 전 **자동 백업**을 만듭니다.
3. **셀 표시 오류 결정론 탐지**(워크북이 열린 동안): 출력 대상 시트에서 `Range.Text`가
   `#` 2개 이상으로만 표시되거나 `#REF!`/`#DIV/0!` 등 수식 오류인 셀을
   **파일·시트·셀주소**까지 탐지합니다.
4. 모든 시트를 **폴더·파일 순서로 단일 PDF에 병합**(폴더·파일 간지 삽입, 제목=폴더명/파일명)
5. 결과물 = **PDF 파일 1개**
6. **출력물 오류 검토**: PDF 텍스트와 렌더링 이미지를 프로그램 로직으로 검토해
   이상과 위치(폴더/파일/시트/페이지)를 리포트로 보고합니다.
   결과는 `출력물 오류 검토결과.xlsx`로도 저장됩니다.

> **`###` 관련 주의:** 위 결정론 탐지는 **셀 자체가 `###`로 표시되는 경우**만 잡습니다.
> 그림/단면도 안의 숫자가 **출력 배율(fit-to-page) 때문에** `###`처럼 찌그러지는 경우는
> 셀 값(예: `0.30`)은 정상이라 이 방식으로 잡히지 않습니다 — 이는 **출력 설정 문제**이며,
> 시각 확인 또는 출력물 오류 검토 결과로 점검하세요.

## 사용 전 준비

1. **Microsoft Excel** 과 **Python** 설치
2. 패키지 설치: `pip install -r tool/requirements.txt`
   (`pywin32`+`pymupdf` 필요)

## 사용법 (CLI)

```bash
python tool/toolruntime.py "수량산출서_루트폴더"
python tool/toolruntime.py "루트" --no-backup     # 원본 백업 생략(주의)
python tool/toolruntime.py --review "루트"        # 루트(또는 루트\_output)의 통합 PDF 검토
python tool/toolruntime.py --review "루트\_output\수량산출서 output.pdf"
```

## 사용법 (GUI)

`tool\run_gui.bat` 더블클릭 → 루트 폴더 선택 → [PDF 생성 시작].
- `원본 백업(권장)` 체크박스
- [PDF 생성 시작] 후 [출력물 오류 검토] 버튼으로 프로그램 내부 검토 실행

## 결과물

모든 산출물은 `루트\_output\` 폴더에 모입니다 (`--out` 지정 시 그 폴더).

- `루트\_output\수량산출서 output.pdf` — **단일 병합 PDF**
- `루트\_output\manifest.json` — 페이지↔폴더/파일/시트 매핑
- `루트\_output\cell_issues_<시각>.json` — 셀 표시 오류(`##` 이상 / `#REF!` / 수식오류) 위치
- `루트\_output\review_<시각>.json` — 프로그램 오류 검토 결과
- `루트\_output\출력물 오류 검토결과.xlsx` — 사용자가 확인할 검토 결과표
- `루트\_output\process_report.json` — 파일별 포함/제외 내역
- `루트_backup_<시각>\` — 원본 백업 (산출물 폴더/PDF/JSON 리포트는 백업에서 제외)

## 중요 주의

- 2단계에서 **원본 엑셀에 "페이지 나누어 미리보기"가 저장**되어 원본이 변경됩니다.
  기본적으로 처리 전 백업을 만들지만, 중요 자료는 별도 백업을 권장합니다.
- **셀 너비/높이·배율은 변경하지 않습니다**(기존 인쇄설정 존중). 인쇄영역이 지정되지 않은
  시트는 Excel 기본값(사용범위 자동)으로 인쇄되어 페이지가 여러 장으로 나뉠 수 있습니다.

## 오류 검토 기준

- Excel 단계: 셀 표시값이 `#` 2개 이상으로만 보이는 셀, `#REF!`, 기타 수식 오류를 탐지합니다.
- PDF 단계: 텍스트 레이어의 `##` 이상, `#REF!`, 빈 페이지, 페이지 하단 표 테두리 누락 의심을 탐지합니다.
- 검토결과의 `출력물 위치`는 최종 병합 PDF 파일 기준 페이지입니다.
- 하단 표 테두리 검토는 렌더링 이미지 기반 휴리스틱이므로 의심 위치를 알려주는 보조 검토로 봐 주세요.
